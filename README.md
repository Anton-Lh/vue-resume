## 个人简历
本项目是基于Vue、VueRouter、Bootstrap开发的一个简单的个人简历。

#### 开发环境
* node
* webpack
* vue-cli

#### 技术栈
* vue 
* vueRouter  
* bootstrap

#### 运行打包
git clone git@github.com:caozongpeng/vue-resume.git
``` bash
# 安装依赖
npm install

# 运行默认是本的localhost:8080
npm run dev

# 打包
npm run build

```
